## Installation
1. Clone the repository ```https://github.com/ubpon/blog-post.git```
2. Run these commands
```php
    cd blog-post // or equivalent cloned name
    composer install
    php artisan key:generate 
    php artisan migrate --seed 
    php artisan test
```
